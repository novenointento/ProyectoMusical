# Project Kickoff Skill

This generic skill helps an AI agent start a brand-new project from this template.

## Use when

- The domain is not implemented yet.
- The team needs to define the first real feature.
- You want to replace placeholder frontend/backend examples.

## Steps

1. Read README.md and the active OpenSpec change.
2. Confirm the feature scope and acceptance criteria.
3. Replace examples in backend, frontend, Prisma, and docs.
4. Mark progress in `openspec/changes/<change>/tasks.md`.
