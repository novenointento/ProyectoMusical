# Frontend Template

Este frontend es una base minima en React para empezar un proyecto nuevo.

## Punto de partida

- `src/App.js`: pantalla inicial de la plantilla
- `src/components/TaskTemplateCard.js`: ejemplo de componente reutilizable
- `src/services/api.js`: ejemplo de llamada al backend
- `cypress/e2e/template.cy.ts`: prueba E2E de ejemplo

## Arranque

```bash
npm install
npm start
```

## Sustituir primero

- Reemplaza la cabecera y las tarjetas de ejemplo por tu primer flujo real.
- Cambia `fetchHealth()` por tus servicios reales.
- Mantén una pantalla minima funcional mientras el producto crece.
